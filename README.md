# Vale do Sol – Jogo Top-Down Estilo Stardew Valley

Projeto desenvolvido em **Unity** com inspiração em *Stardew Valley*, apresentando mecânicas de **agricultura, coleta de recursos, combate e construção**.

<img width="548" height="309" alt="image" src="https://github.com/user-attachments/assets/5de52434-12e3-4ce7-9c03-d3cea8643b5a" />

---

## Objetivo do Projeto
Integrar diferentes sistemas de gameplay em um mesmo ambiente, combinando **exploração, agricultura, coleta, combate e construção**, garantindo que cada mecânica funcione de forma coerente e interligada.

---
